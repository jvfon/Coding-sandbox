# Types - Intro

## Primitive type
- String  
Good for holding text.  
3 types:  
    - Single quotes: `'chair'`
    - Double quotes: `"chair"`
    - Back ticks: `` `chair` ``

- Number  
Can be integers, decimals, etc.  

- Boolean  
true or false  

- Null  
- Undefined  
- Symbol  

## The "special" type
- Object  
Everything in JS is an object
